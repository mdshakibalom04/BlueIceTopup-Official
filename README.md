# BlueIceTopup Official Android App

Package: `com.blueicetopup.official`  
Website: `https://blue-ice-topup.web.app/`

## GitHub থেকে APK Build

1. এই ZIP extract করুন।
2. GitHub repository `BlueIceTopup-Official` খুলুন।
3. **Add file → Upload files** চাপুন।
4. Extract করা `BlueIceTopup_Official_Android` ফোল্ডারের ভেতরের সব ফাইল/ফোল্ডার আপলোড করুন।
5. **Commit changes** চাপুন।
6. Repository-এর **Actions** tab খুলুন।
7. **Build Android APK** workflow খুলুন।
8. **Run workflow → Run workflow** চাপুন।
9. Build সফল হলে **Artifacts** থেকে `BlueIceTopup-Official-APK` download করুন।
10. ZIP খুলে `app-debug.apk` install করুন।

এটি installable debug APK তৈরি করবে। Play Store-এর জন্য পরে signed release AAB/APK তৈরি করতে হবে।
